# Home

Welcome to the Red Reactor wiki!

If you haven't already got yourself a Red Reactor, they can be found on [the Red Reactor website](https://theredreactor.com) — thanks to [Scally-H](https://github.com/Scally-H) for designing and building us all a high quality UPS!

## Setup

Choose the guide that matches your environment:

* [Home Assistant OS](Home-Assistant-OS) — Install via the dedicated Red Reactor add-on
* [Ubuntu / Standard Linux OS](Linux) — Run as a systemd service with a dedicated user
* [Docker](Docker) — Run as a container with I2C device passthrough
* [PyPI](PyPI) — Install directly with `pip` for a quick manual setup
